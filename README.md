# FreeFlow 💸

A modern personal finance and cashflow forecasting app built with **React, Vite, and Tailwind CSS**.

FreeFlow is designed to help users manage income, expenses, bills, savings, debt, and future cashflow in one place — with a strong focus on **pay-cycle planning and financial visibility**.

---

## Features (Planned)

### Dashboard
- Financial overview
- Total balances
- Monthly income vs expenses
- Disposable income tracking
- Quick financial insights

### Accounts
- Transaction accounts
- Savings accounts
- Credit cards

- Investment tracking

### Income & Pay Cycles
- Weekly, fortnightly, monthly income
- Multiple income sources
- Salary tracking

### Expenses & Bills
- Recurring expenses
- Upcoming bills
- Categories & tracking
- Budget visibility

### Savings Goals
- Goal tracking
- Progress monitoring

### Debt Tracking

### Forecasting

- Cashflow simulation
- Pay-cycle forecasting
- Expense modelling

### Reports

- Financial summaries

```bash

```

### Build for production

```bash
npm run build
```
